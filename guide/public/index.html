After Purchasing From Metamask

 
  1) Click On The Contract Address (Then Copy the Contract Address(Address Hash) From The GoChain Explorer)




  2)Look at the Total supply Number (N) ,the last (N) and Below is Ur NFTs (Copy the TokenIDs somewhere..Or Remember It)




  3)Click On Wallet - Open Wallet (With Ur Metamask Private Keys )




  4) Go To "Use Contract" Tab and Paste The GoBears Contract Address

5)Choose Contract Function: safeTransferFrom (from,to,tokenid) 





  6)Fill In The Details And Click Submit 
7)From(Metamask Wallet address) to (Zeromint Wallet Address ) token id (Ur NFTs Token ID) 






  8)Then Go To Ur Zeromint Account and visit The GoBears Collection Page and Click Import NFT (put ur Token ID ) 



  9)Thats it ! Now you can View Ur GoBears on Zeromint.
